# Network Traffic Analysis lab 

## Objective
The Network Traffic Analysis Lab (Webstrike Lab on Cyberdefenders) is a challenge that simulates a real world incident involving a suspicious web shell. The task involves analyzing network logs and identifying malicious activities from a forensic standpoint.

### Skills Learned
- Advanced understanding of Packet Capture (PCAP) Analysis.
- Proficiency in analyzing and interpreting network logs.
- Ability to track web shells with persistent access via url patterns.
- Development of attention to details and problem-solving skills in cybersecurity.

### Tools Used
- Network analysis tools; Wireshark for examining network traffic.

## Steps
- Identify unusual IP addresses.
- Check for unauthorized or suspicious file upload.
- Identify where uploaded files(if any) are saved on the network.
- Identify ports through which attacker got access to the network.
- Check for data exhilitration and identify which data was exhiltrated.
